    2. Zaokrąglenie narożników obszarów bloków np. DIV
    Zaokrąglenie górnego-lewego narożnika: selektor { border-top-left-radius: zaokrąglenie }
    Zaokrąglenie górnego-prawego narożnika: selektor { border-top-right-radius: zaokrąglenie }
    Zaokrąglenie dolnego-prawego narożnika: selektor { border-bottom-right-radius: zaokrąglenie }
    Zaokrąglenie dolnego-lewego narożnika: selektor { border-bottom-left-radius: zaokrąglenie }

  2. Zastosowanie oraz uwagi dotyczące zaokrąglania narożników

    • We wszystkich przypadkach selektorem może być dowolny znacznik np.[p, h1, div
    • "Zaokrąglenie" określa promień wyrażony w jednostkach długości albo w procentach
    • Co ciekawe, element wcale nie musi mieć obramowania, aby miał zaokrąglone narożniki.
      Wystarczy jeśli będzie miał ustalone jakieś tło, aby efekt był widoczny na ekranie.
    • Tekst wewnątrz elementu nie zostanie zawinięty do zaokrąglonych rogów. Jeśli promień zaokrąglenia będzie zbyt duży, tekst wyjdzie na zewnątrz.

  3. Przykład

    • border-top-right-radius: 10px
